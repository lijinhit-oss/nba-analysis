# 🏀 NBA 每日分析报告自动推送系统

每天自动抓取 Pinnacle 赔率、Polymarket 预测市场价格和 ESPN 赛程/战绩数据，计算去水胜率、偏差信号与策略建议，生成可视化 HTML 报告并自动发布到 GitHub Pages。

---

## 项目介绍

本系统通过以下数据源对当日 NBA 比赛进行量化分析：

| 数据源 | 用途 |
|--------|------|
| **TheOddsAPI（Pinnacle）** | 获取 Pinnacle 赔率，计算去水真实胜率 |
| **Polymarket** | 获取去中心化预测市场的主队胜率 |
| **ESPN** | 获取今日赛程、球队战绩和分区排名 |

分析维度包括：
- **Pinnacle 去水胜率**：去除庄家水钱后的真实概率估算
- **Polymarket vs Pinnacle 差值**：识别市场定价偏差
- **偏差信号**：🔴 重大偏差 / 🟡 中等偏差 / 🟢 基本一致
- **策略类型**：策略一（势均力敌）/ 策略三（强弱悬殊）
- **球队动机评估**：结合战绩和分区排名判断出战意愿
- **优先级排序**：★★★ > ★★☆ > ★☆☆

## 部署步骤

### 第一步：Fork 本仓库

点击页面右上角的 **Fork** 按钮，将本仓库复制到你的 GitHub 账号下。

---

### 第二步：注册 TheOddsAPI 获取免费 Key

1. 访问 [https://the-odds-api.com/](https://the-odds-api.com/) 注册账号
2. 免费套餐每月提供 **500 次**请求额度（日常使用绰绰有余）
3. 注册后在控制台页面复制你的 **API Key**

---

### 第三步：配置 GitHub Secret

1. 进入你 Fork 后的仓库页面
2. 点击顶部导航栏的 **Settings**
3. 左侧菜单选择 **Secrets and variables → Actions**
4. 点击 **New repository secret**
5. 填写以下信息：
   - **Name**：`ODDS_API_KEY`
   - **Secret**：粘贴你在第二步获取的 API Key
6. 点击 **Add secret** 保存

---

### 第四步：开启 GitHub Pages

1. 在仓库 **Settings** 页面，左侧菜单选择 **Pages**
2. 在 **Source** 下拉菜单中选择分支 **`gh-pages`**
3. 目录保持默认 `/ (root)`
4. 点击 **Save**

首次部署完成后，你的报告将发布到：
```
https://<你的用户名>.github.io/<仓库名>/
```

## 自动运行说明

工作流配置为每天 **UTC 08:00（北京时间 16:00）** 自动触发：

```yaml
on:
  schedule:
    - cron: '0 8 * * *'  # UTC 08:00 = 北京时间 16:00
```

每次运行后，最新的 `output/index.html` 将自动推送到 `gh-pages` 分支并更新 GitHub Pages 页面。

> **注意**：GitHub Actions 的定时任务在高负载时可能有 5-15 分钟延迟，属正常现象。

---

## 手动触发方法

如需随时手动刷新数据，无需等待定时任务：

1. 进入仓库的 **Actions** 标签页
2. 左侧列表选择 **NBA Daily Report**
3. 点击右侧 **Run workflow** 按钮
4. 在弹出框中点击绿色 **Run workflow** 确认
5. 等待约 1-2 分钟，工作流运行完成后刷新 Pages 链接即可查看最新报告

## 策略说明

### 策略一：势均力敌型

**适用条件**：Pinnacle 去水胜率差值 **< 15%**

两队实力相近，比赛结果不确定性较高。此类比赛重点关注：
- 赛前盘口的异动（资金流向暗示内部信息）
- Polymarket 与 Pinnacle 偏差是否随时间扩大
- 伤病、主客场、背靠背等临场因素

操作建议：观望为主，等待偏差信号达到中等及以上再考虑介入。

---

### 策略三：强弱悬殊型

**适用条件**：Pinnacle 去水胜率差值 **≥ 15%**

强队对阵弱队，结果较为明朗。此类比赛关注点：
- Polymarket **低估**强队胜率时：存在低风险价值机会
- Polymarket **高估**弱队胜率时：警惕跟风情绪，避免追冷门
- 弱队球队动机评估为"弱"时：摆烂概率上升，强队覆盖优势更大

操作建议：若偏差信号为🔴重大偏差且方向合理，可作为参考依据。
